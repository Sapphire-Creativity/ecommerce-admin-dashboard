## ✨ Features

- **Dashboard Homepage**
  - Line chart data visualization  
  - Overview cards with real-time data  

- **Products Page**
  - Add, edit, and delete products  

- **Orders Page**
  - Add/edit orders  
  - Select products from existing catalog  

- **Customers Page**
  - View customer information  

- **Profile Page**
  - View and edit admin profile details
